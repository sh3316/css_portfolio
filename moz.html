Project: MozArt Machine – Marker Sketch to Music Translation

Designed and implemented a system that converts marker sketches into melodies played on a glockenspiel. Marker sketches are drawn on card machine receipt-style paper strips, which are fed into the MozArt machine using stepper-driven rollers. A camera module captures sequential snapshots of the strip, stitching them into a complete image. Both the stepper motor and camera are controlled by a Raspberry Pi 3, which regulates motor speed and camera shutter timing.

The algorithm for translating sketches into music operates in four key steps:

1. Calibration: A threshold value is calculated by analyzing a blank section of paper, identifying the minimum pixel brightness to detect marker lines accurately.
2. Image Processing: At each time step, a snapshot is cropped to its central column and converted to a binary image based on the threshold, with darker pixels representing lines.
3. Music Mapping: The processed image is scaled to an 8x1 Boolean vector, where each value corresponds to one of the eight glockenspiel notes to be played.
4. Melody Optimization: Adjacent notes are removed to ensure a harmonious melody, leveraging the D Major scale for a pleasant, though simple, musical output.
This innovative project bridges image processing and music generation, transforming visual sketches into engaging auditory experiences.

Topics: Image Processing, Embedded Systems, Music Generation, Algorithm Design, Raspberry Pi Applications
