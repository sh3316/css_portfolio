Project 1: Custom Evaluation Function for Isolation Game

Developed an evaluation function tailored for an Isolation game environment. This function calculates scores based on available moves for each player, aiming to enhance AI decision-making by prioritizing positions with greater strategic value. The green piece represents the human player, while the red piece is the AI agent. Implemented in Python, this project required optimizing decision points to improve gameplay strategies.

Topics: Game Theory, Alpha-Beta Pruning, Iterative Deepening Algorithm

Project 2: Gaussian Mixture Model for Image Segmentation

Utilized Gaussian Mixture Models to perform unsupervised image segmentation, enhancing pixel clustering and component likelihood calculations. This involved refining initial parameters using k-means and creating a convergence function to optimize the segmentation accuracy, resulting in a segmented output that represents distinct image components based on intensity.

Topics: Gaussian Mixture Models, k-means, Convergence Functions

Project 3: Bidirectional Pathfinding Algorithm

Engineered a pathfinding algorithm capable of efficiently finding paths in graph structures by implementing bidirectional search techniques. This project incorporated the calculation of cumulative costs and optimal path intersections between nodes, enhancing the speed and accuracy of navigation within complex graph environments by leveraging a* search combined with bidirectional search.

Topics: Graph Theory, Bidirectional Search, Pathfinding

Project 4: Decision Tree and Random Forest Implementation

Developed a custom Decision Tree and Random Forest model, along with a set of evaluation metrics including precision, recall, accuracy, and confusion matrix computation. This project emphasized creating efficient data processing functions with vectorized numpy operations to handle large datasets. Additionally, I built functions for Gini impurity and gain calculations, essential for decision tree splits, and implemented a k-fold cross-validation mechanism to evaluate model performance across multiple folds.

Topics: Machine Learning, Decision Trees, Random Forests, Numpy Vectorization, Evaluation Metrics

Project 5: Bayesian Network for Security System and Game Network Simulation

Designed a Bayesian Network to model a security system with conditional probability distributions for key components and interactions. The project also implemented Gibbs and Metropolis-Hastings sampling methods for inference on a game network to compare convergence and efficiency. By creating functions for sampling, calculating posterior distributions, and setting up custom conditional probability tables, this project demonstrates advanced skills in probabilistic graphical models and Bayesian inference using the pgmpy library.

Topics: Bayesian Networks, Gibbs Sampling, Metropolis-Hastings Algorithm, Conditional Probability Distributions
