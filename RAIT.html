Project 1: Kalman Filter Prediction and Laser Defense System for Meteorites

Designed a system to predict and intercept falling meteorites using Kalman Filters in a noisy observation environment. The assignment required precise estimations of meteorite trajectories despite position noise, limited computational resources, and strict timing constraints to predict and intercept within milliseconds. Complex restrictions, such as maintaining a high accuracy threshold across consecutive timesteps and optimizing angle adjustments within limited aiming ranges, added to the challenge. The laser turret's defense mechanism calculated optimal firing angles and accurately targeted meteorites to prevent ground impact.

Topics: Kalman Filters, State Estimation, Noise Reduction, Robotic Defense Systems

Project 2: Particle Filter-Based Satellite Localization and Targeted Signal Transmission

Developed a particle filter system to accurately estimate the satellite’s position within a noisy, simulated solar system. The project required implementing core particle filter techniques, including initialization, importance weighting, resampling, and fuzzing, to handle the complexities of gravitational noise and limited observational data. Each particle’s weight is calculated using Gaussian probability based on gravitational measurements, ensuring a robust estimate. A custom resampling wheel retains the most probable particles, while a fuzzing algorithm prevents premature convergence by spreading particles over a broader area. In the second phase, the satellite's position estimate is used to dynamically compute an optimal SOS transmission angle to the home planet, improving rescue signal targeting under strict resource constraints. The Red Dot is the target, Green Dots are the peripheral objects, Light Blue and White Particles are estimates of the code.

Topics: Particle Filter, Resampling Wheel, Gaussian Weighting, Fuzzing, Probabilistic Localization, Signal Angle Calculation

Project 3:SLAM and Path Planning for Indiana Drones

Developed a Simultaneous Localization and Mapping (SLAM) system and a path-planning algorithm for a drone navigating a jungle environment with unknown obstacles. The SLAM component estimates the positions of the drone and trees based on noisy sensor measurements, while the path-planning component guides the drone towards a treasure, avoiding obstacles. Implemented robust matrix operations, Cholesky decomposition, and probabilistic algorithms to optimize localization and navigation accuracy. The project required efficient handling of sensor noise and real-time updates to the drone’s belief state. Red Path is intended path and Blue Path is actual path that drone moved. The small gray dots are objects' estimated center through detection.

Topics: SLAM, Path Planning, Matrix Operations, Bayesian Inference, Sensor Fusion

Project 4: Warehouse Robot Navigation and Delivery Optimization

Implemented algorithms to enable a robot to navigate through a grid-based warehouse environment to retrieve boxes and deliver them to a drop zone. The robot follows specific paths using A* search for efficient navigation, considering movement costs and constraints. The code adapts to various configurations in Parts A, B, and C, each with increasing levels of complexity. Movement penalties for illegal paths and stochastic actions were also integrated, ensuring the robot optimally completes tasks under realistic conditions.

Topics: A* Search, Priority Queue Management, Heuristic Pathfinding, Dynamic Programming
