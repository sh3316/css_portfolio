Project 1: Object Removal Using Patch Matching and Confidence Propagation

Implemented an advanced inpainting algorithm to remove objects seamlessly from images by utilizing patch matching and confidence propagation. The process begins by identifying the target area and examining surrounding textures and patterns. Using an iterative patch selection technique, the algorithm finds and fills gaps with patches that blend naturally, preserving structural details like edges and gradients along object boundaries. Confidence propagation is applied to ensure the filled area aligns smoothly with surrounding pixels, producing a highly realistic inpainting effect. This approach is especially suited for high-quality image restoration and object removal in visual applications.

Topics: Patch Matching, Confidence Propagation, Boundary Detection, Image Restoration

Project 2: Seam Carving for Content-Aware Image Resizing

Developed a comprehensive image resizing and object removal solution using seam carving. This project involved implementing an energy-based method for adaptively resizing images while preserving key content areas. The algorithm calculates energy maps and removes or inserts seams—paths of least importance—based on dynamic programming techniques. Object removal is also supported, with a specified mask guiding the elimination of unwanted regions by iteratively removing seams, ensuring minimal distortion in surrounding areas. The system is optimized for both performance and quality, making it ideal for responsive image resizing in UI/UX design and photography.

Topics: Seam Carving, Energy Map Calculation, Dynamic Programming, Content-Aware Scaling

Project 3: Multi-Resolution Pyramid Blending for Image Mosaics

Implemented an image blending algorithm using Gaussian and Laplacian pyramids to create seamless transitions between images. This project involved constructing Gaussian and Laplacian pyramids, blending images with a Gaussian mask at multiple resolutions, and collapsing the pyramids to form a single blended output. Key tasks included custom implementations for image reduction, expansion, and convolution without OpenCV’s built-in pyramid functions. The work demonstrates advanced techniques in image processing, with practical applications in image mosaics and photo manipulation.

Topics: Image Processing, Gaussian and Laplacian Pyramids, Convolution, Blending

Project 4: HDR Image Processing and Radiance Mapping

Created an HDR imaging pipeline that generates high dynamic range (HDR) images from multiple exposure photos, based on the Debevec and Malik method. The project focused on simulating real-world lighting by computing radiance maps, tone mapping, and histogram equalization for a balanced output. To achieve this, I implemented key functions such as intensity sampling, response curve computation, radiance map generation, and histogram equalization without relying on built-in functions. The final result enhances detail in both shadows and highlights, offering visually rich, realistic images ideal for professional photography and visual effects.

Topics: HDR Imaging, Radiance Mapping, Exposure Fusion, Histogram Equalization
